# Henkilotietorekisteri
 C# kirjoitettu henkilötietorekisteri. Tallentaa tiedot salaamattomana tekstitiedostoon. Tietojen muuttaminen luo merkinnän lokiteidostoon.

 Tehty ensimmäisen vuoden kurssin lopputyönä.
